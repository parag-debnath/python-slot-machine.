# 🎰 Python Slot Machine

A text-based slot machine game implemented in Python. This project demonstrates core programming concepts including user input validation, nested loops, dictionary manipulation, and modular function design.

## 🚀 Features
- **Dynamic Betting:** Choose the number of lines to bet on and the amount per line.
- **Balance Management:** Includes checks to ensure the player has sufficient funds before spinning.[cite: 1]
- **Randomized Logic:** Uses a weighted symbol system to simulate real slot machine probabilities.[cite: 1]
- **Winning Detection:** Automatically calculates winnings based on matching symbols across horizontal lines.[cite: 1]

## 🛠️ Concepts Applied
- **Functions & Modularity:** Logic is split into reusable functions for spinning, checking winnings, and handling input.[cite: 1]
- **Input Validation:** Prevents the program from crashing due to invalid inputs or insufficient funds.[cite: 1]
- **Data Structures:** Uses lists and dictionaries to manage symbol counts and values.[cite: 1]
- **String Formatting:** Clean user interface using Python f-strings.[cite: 1]

## 🎮 How to Play
1. **Deposit:** Enter the amount of money you want to start with.[cite: 1]
2. **Select Lines:** Choose how many lines (1-3) you want to bet on.[cite: 1]
3. **Place Bet:** Enter the amount you want to bet per line.[cite: 1]
4. **Spin:** The machine will generate a grid. If the symbols align, you win![cite: 1]
5. **Repeat or Quit:** Keep playing until you run out of money or choose to exit.[cite: 1]
